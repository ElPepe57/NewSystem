Por qué se mantiene el avatar dropdown
  • Atajo rápido cuando el sidebar está colapsado (modo icon-only)
  • Acceso cross-page · siempre visible en topbar
  • Convención web universal · users esperan encontrarlo ahí
  • Espacio natural para "Cerrar sesión" y configuraciones de cuenta
Diferencia funcional vs sidebar
Sidebar Mi espacio Navegación principal · siempre visible · grupo completo
Avatar dropdown Atajo + session info + acciones de cuenta + logout
Decisión clave
"Cerrar sesión" vive en ambos lugares · sidebar (Mi espacio bottom) y dropdown. Es la acción más buscada y NO hay riesgo de duplicación (logout es idempotente).

Reglas de active state
  • 1 solo item activo a la vez · siempre el que matchea la ruta actual
