Resultado · el sidebar queda cromáticamente coherente

Al entrar a un módulo, su color de chrome coincide con el color de su grupo en el menú. El usuario aprende "violet = mi gente, teal = mi plata".

Finanzas y Contabilidad · teal
Finanzas ya teal ✓
Gastos ya teal ✓
Contabilidad cambia → teal
Equipo · violet
Inversionistas ya violet ✓
Usuarios cambia → violet
Planilla cambia → violet

Importante · qué NO se toca

El cambio es solo el chrome de identidad (6 puntos). El color semántico de datos se conserva 100%:

KPIs verde=positivo / rojo=negativo
Donuts de Balance (teal/rose/indigo)
Badges de estado (pagado/pendiente)
Sparklines y gráficos
Color por bloque/origen (azul/morado/ámbar)
Insights y banners de alerta
