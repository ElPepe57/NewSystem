Canon v8.0 aplicado

N1Tinte semántico por gráfica · purple ejecutivas · teal operativas · amber tácticas
N2Cards gradient + ring colored para cada KPI/gauge

Deudas técnicas implementación S4

DEUDA-LIBRERIA-GRAFICA: decidir Recharts vs Chart.js vs Visx para gráficos complejos (Sankey · Cohort · Bridge). SVG inline ok para gauges simples.
DEUDA-FISCAL-FUTURO: G1.a sin impuestos · activar G1.c cuando se integre módulo SUNAT.
DEUDA-CALCULO-BURN-RATE: definir qué cuentas/categorías cuentan como "fijos" para el burn rate (sueldos, alquiler, SaaS recurrentes).
DEUDA-COHORT-DSO: requiere tracking de fecha-primera-venta por cliente · auditar si ya existe.
DEUDA-EBITDA-DRIVERS: identificar drivers de variación (volumen vs precio vs costos vs FX) requiere algoritmo · puede simplificarse en MVP.
