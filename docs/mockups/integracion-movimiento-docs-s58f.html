Movimientos sin doc origen requieren motivo obligatorio + autor

Es la única forma de mantener trazabilidad cuando no hay un doc operativo detrás. Coordinado con system-auditor · cualquier movimiento huérfano sin motivo se marca como "no conformidad" en auditorías post go-live.

Decisiones declaradas en este mockup

D-INT-MOV-1
Doc origen como card prominente. No se entierra en un campo · se muestra como CTA grande con color del módulo (fucsia=venta, sky=OC, orange=gasto, indigo=envío). Click → abre el doc origen en el modal correspondiente.
D-INT-MOV-2
Hermanos del split como sección dedicada. Cuando un movimiento es 1 de N líneas de un split, los hermanos se listan con CTA y se muestra el cuadre total. El propio movimiento se destaca con badge "tú estás aquí" · pattern de orientación.
D-INT-MOV-3
Caso múltiples docs origen agrupados. En liquidación de caja recaudadora, 1 movimiento = N ventas liberadas. Se muestra tabla agrupada con cada doc + monto liberado + total. Click sobre cada fila abre el doc específico.
D-INT-MOV-4
Conversiones · diagrama N:1 o 1:N. Liquidación de 3 cajas → 1 ingreso BCP es N:1. Movimientos hermanos se listan con cuadre numérico ("S/3,920 + S/1,315 + S/220 = S/5,455 ✓"). Pattern visual de "antes / flecha / después".
D-INT-MOV-5
Impacto si elimino · prevención preventiva. Antes de eliminar un movimiento, se muestra exactamente qué saldos cambian, qué docs origen quedan afectados, y qué hermanos se huerfanan. Reduce errores destructivos.
D-INT-MOV-6
Auditoría como timeline inline. Cuándo se creó, quién, en qué contexto · qué eventos futuros (conciliación, liquidación) están agendados. No requiere abrir otra pantalla.
D-INT-MOV-7
Movimiento huérfano legítimo · 3 casos. Ajuste manual (cuadre) · transferencia interna entre cuentas propias · aporte de capital. Cada caso usa pattern visual distinto pero con campo común: motivo obligatorio. Sin motivo → no se permite guardar.
D-INT-MOV-8
Coordinación con system-auditor. Movimientos huérfanos sin motivo aparecen como hallazgos en auditorías mensuales · cierre del círculo entre Tesorería operativa y compliance interno.
