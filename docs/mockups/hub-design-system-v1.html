Hub Kit · el sistema del que nace todo módulo

Un set de building blocks canónicos que ensamblan cualquier módulo del ERP. En vez de re-diseñar y revisar el shell cada vez (top-bar, header, KPIs, tabs, body, estados…), un módulo nuevo se compone desde el kit y nace ya consistente, responsive y pixel-perfect. Imposible desviarse: la consistencia vive en el componente, no en la disciplina.

El problema que resuelve: hoy cada módulo re-implementa el shell a mano, en momentos distintos → derivan max-w, padding, orden, recuadros anidados. Cada módulo nuevo cuesta rondas de revisión visual. Con el kit: el módulo nace correcto de fábrica.
Principio · composición sobre repetición
1 fuente de verdad
El shell vive en componentes `@hub`. Cambiar el canon = editar 1 lugar, se propaga a todos.
Consistencia blindada
max-w, padding, spacing, orden y estados los resuelve el kit. El módulo no puede inventar.
Módulo nuevo = ensamblar
Un módulo nuevo se arma componiendo blocks. Cero rondas de revisión de shell.

Ensamblaje · cómo nace un módulo desde el kit
// Gastos (Layout A · con sidebar) — TODO el shell en 1 ensamblaje
<HubShell>
  <HubTopBar modulo="Gastos" leaf={tab} onVolverModulo={...} color="teal" />
  <HubHeader icon={Receipt} title="Gastos" subtitle={...} actions={...} />
  <HubKpiStrip kpis={kpiData} miniStats={miniStats} />
  <HubTabs tabs={TABS} active={tab} onChange={setTab} color="teal" />
  <HubBody aside={<SidebarGastos/>}>
    {tab === 'resumen' && <ResumenGastos/>}
    {tab === 'movimientos' && <ListaGastos/>}
    {tab === 'analisis' && <AnalisisGastos/>}
  </HubBody>
</HubShell>

// Contabilidad (Layout B · sin sidebar) — mismo kit, sin prop aside
<HubShell>
  <HubTopBar modulo="Contabilidad" leaf={tab} color="purple" />
  <HubHeader ... />  <HubKpiStrip ... />  <HubTabs ... />
  <HubBody> {/* sin aside → full-width */}
    {tab === 'balance' && <BalanceGeneral/>}
  </HubBody>
</HubShell>

El módulo solo aporta su data y su contenido. Todo el chrome (recuadro, breadcrumb, padding, max-w, responsive, estados) lo da el kit.
