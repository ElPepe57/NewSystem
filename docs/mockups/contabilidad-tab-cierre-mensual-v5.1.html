¿Qué pasa cuando cerrás un mes?
  • Se bloquean modificaciones a movimientos del período (ventas/compras/gastos)
  • Se genera snapshot inmutable de Balance + P&L + Indicadores
  • Quedan registrados quién cerró + cuándo (audit trail)
  • Para corregir algo: re-abrir el período (acción auditada) · hacer cambios · volver a cerrar
  • La utilidad del período se traslada automáticamente a "Utilidades acumuladas" en el siguiente Balance
